What are the types of variables that C offers

There are two basic types that you will learn today integers and floating point numbers.

Integers

These types of variables can only hold integers as the name might suggest. The way that the number is stored is much like how you might represent a number in a decimal number system but computers can only store 1s and 0s so it's the number system is binary.

What are the different Integer types

name Bits and bytes range other names
signed char 8-bits or a byte -128 to 127 char depending on systems
unsigned char 8-bits or a byte 0 to 255 char depending on systems
short 16-bits or 2-bytes -32,768 to 32,767 short int signed short int or signed short
unsigned short 16-bits or 2-bytes 0 to 65,535 unsigned short int
int 32-bits or 4-bytes -2,147,483,648 to 2,147,483,647 signed int and signed
unsinged int 32-bits or 4-bytes 0 to 4,294,967,295 unsigned

After looking at the table above you might have realized something that there are 4 different main names char, short, int, and long. Then the signed (the ones with no prefix) and unsigned prefixes modify the range of the numbers that can be stored in the types.

Real number

Used when you want to have a decimal point in the number meaning it's a real number.

What are the types

name Bits and bytes
float 32-bits or 4-bytes
double 64-bits or 8-bytes
long double same as the double or as large as 10-16 bytes

The table above doesn't have signed and unsigned types that's because they aren't offered. Also that there float, double, and long double like there were short, int, and long for the integer types

Things to watch out for

Because there is a finite amount of bits that the float, double, and long double types store the number isn't always fully precise, this means you should avoid using the "==" operator since it might look like the numbers are the same they might be a different of .00001 that the number picks up on

How to print variables

If you looked at it you might see that there is a weird simple "%d", this means that the first argument after print text is a int and the value should be printed. If the "%d" was the second Specifier then the variable that should be printed should be the second after the print text.
