First you go to the cs50 releases. Then download the latest source code in zip. Open up your file manager add go to your Downloads folder then shearch for libcs50, unpack that. After unpaking the zip file you should see a folder you should open that folder in the terminal. After that is done execute this code:

Getting Integers from the User

We will be using the cs50 library that we just installed to do this, most inportantly the get_int() function

What it looks like

                    
get_int("string you what to specify the input");

Return values

This function only returns an int that the user gave

How it might be used

for example it might be used to get a number between 1 and 10

                    
// including stdio and cs50 libraries so that the program has access to them
#include <stdio.h>
#include <cs50.h>

// This is the first function that runs
int main(void){
    // the integer variable number is assigned the value of 0 (place holder)
    int number = 0;

    // Then the fancy do while loop loops untill the while condition is met
    do{
        // Gets an int from the user
        number = get_int("give me a number between 1 and 10 inclusive: ");
    }
    while (number < 1 || number > 10);

    // Prints the number provided by the user
    printf("your number was %i\n", number);
}
